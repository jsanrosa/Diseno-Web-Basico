Creación de Layouts básicos

  • Tres bandas horizontales
  • Tres bandas verticales
  • Cabecera, footer y mitad dividida en tres bandas verticales

1- Estructura básica de un documento HTML

En primer lugar tenemos que tener clara la estructura básica de una página HTML



    <!DOCTYPE html>


    <html>


        <head>


        /<head>   
         

        <body>


        /<body>   
         

    /<html>

2- ¿Qué información va en <head>?

En el <head> se escribe información (metadatos) para ser utilizada por los buscadores y los navegadores. Por ejemplo, el título de la página, el autor, las palabras clave, etc...

3- ¿Qué información va en el <body>?

Sólo puede haber un elemento <body> en un documento HTML. Contiene la información que se quiere mostrar en la página web.

4- ¿Cómo hacer para dividir el espacio de una página web en varias partes?

En primer lugar hay que agrupar todos aquellos elementos que tengan que ir juntos y a los que se les va a aplicar un estilo común. Todas las páginas web tienen al menos las siguientes divisiones:

  1. Una cabecera que suele contener el LOGO y un menú principal
  2. Una zona central que contiene
  3. Un footer o pie de página que contiene información como el copyright e información sobre el autor o la compañía.

5- ¿Cómo agrupo elementos para darles un estilo común?

La manera más habitual es la utilización de un elemento <DIV>. Éste solo sirve de "caja" para agrupar un contenido. Utilizando CSS podemos darle un formato común a todos los elementos DIV. Si les aplicamos una clase o un identificador podemos particularizar sus propiedades de forma que la cabecera tenga un formato distinto que el footer, por ejemplo.

6- ¿Porqué en esta página se ve una zona en color beige debajo del footer?

He dejado esa zona a propósito. Es el fondo (background) del body. Para que no ocurra hay que ajustar bien las propiedades de ancho (width) y alto (height) de cada DIV. La suma de alturas de todos los DIV debe sumar 100 o más.
